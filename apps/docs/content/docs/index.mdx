## Installation

```bash
pnpm add @appshell/react framer-motion
```

### Peer dependencies

`@appshell/react` requires the following peer dependencies:

| Package | Version |
| --- | --- |
| `react` | 18+ |
| `react-dom` | 18+ |
| `framer-motion` | 11+ |
| `tailwindcss` | 4+ |

## Quick start

Wrap your application in `AppShell` and compose the layout with `Header`, `Content`, and `Footer`:

```tsx
"use client";

import {
  AppShell,
  Header,
  Content,
  Footer,
  FooterItem,
} from "@appshell/react";
import { Home, Search, User } from "lucide-react";

export default function App() {
  return (
    <AppShell safeArea>
      <Header
        behavior="fixed"
        theme="light"
        logo={<span className="font-bold">MyApp</span>}
      />

      <Content>
        <div className="p-4">
          <h1>Hello World</h1>
        </div>
      </Content>

      <Footer variant="tab-bar" behavior="auto-hide">
        <FooterItem
          icon={<Home className="size-5" />}
          label="Home"
          active
        />
        <FooterItem
          icon={<Search className="size-5" />}
          label="Search"
        />
        <FooterItem
          icon={<User className="size-5" />}
          label="Profile"
        />
      </Footer>
    </AppShell>
  );
}
```

## Imports

All components and hooks are exported from the package root:

```ts
import {
  AppShell,
  Header,
  Content,
  Footer,
  FooterItem,
  SafeArea,
  AppShellProvider,
  useAppShell,
  useScrollDirection,
  useSafeArea,
} from "@appshell/react";
```

Type exports are also available:

```ts
import type {
  HeaderBehavior,
  HeaderTheme,
  FooterVariant,
  FooterBehavior,
  FooterPosition,
  SafeAreaEdge,
} from "@appshell/react";
```
