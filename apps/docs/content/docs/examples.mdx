The examples app at `apps/examples` contains five complete demos. Each demonstrates a different combination of header behavior, footer variant, and theme.

## Social App

An Instagram/Twitter-style feed with a reveal-on-scroll header and auto-hiding tab bar.

- **Header**: `behavior="reveal-nav"`, `theme="light"`, with search row
- **Footer**: `variant="tab-bar"`, `behavior="auto-hide"`

```tsx
<Header
  behavior="reveal-nav"
  theme="light"
  logo={<span>socialapp</span>}
  searchContent={<input placeholder="Search..." />}
/>
<Footer variant="tab-bar" behavior="auto-hide">
  <FooterItem icon={<Home />} label="Home" active />
  <FooterItem icon={<Search />} label="Explore" />
  <FooterItem icon={<User />} label="Profile" />
</Footer>
```

## E-commerce

A Shopify-style storefront with a fixed header (including context and search rows) and a floating cart button.

- **Header**: `behavior="fixed"`, `theme="light"`, with title, subtitle, and search
- **Footer**: `variant="floating"`, `position="right"`

```tsx
<Header
  behavior="fixed"
  theme="light"
  logo={<span>ShopNow</span>}
  title="New Arrivals"
  subtitle="Spring Collection"
  searchContent={<input placeholder="Search products..." />}
/>
<Footer variant="floating" position="right">
  <button className="size-14 rounded-full bg-primary shadow-xl">
    <ShoppingCart />
  </button>
</Footer>
```

## Messaging

A WhatsApp-style chat with a static primary header and a mini input bar at the bottom.

- **Header**: `behavior="static"`, `theme="primary"`, with contact info in logo
- **Footer**: `variant="mini"`

```tsx
<Header
  behavior="static"
  theme="primary"
  logo={
    <div className="flex items-center gap-3">
      <ArrowLeft />
      <span>Jane Smith</span>
    </div>
  }
/>
<Footer variant="mini">
  <input placeholder="Type a message..." />
  <button><Send /></button>
</Footer>
```

## Music Player

A Spotify-style player with a reveal-all header and a mini now-playing bar. Uses the dark theme.

- **Header**: `behavior="reveal-all"`, `theme="dark"`
- **Footer**: `variant="mini"`, with custom dark styling

```tsx
<Header
  behavior="reveal-all"
  theme="dark"
  logo={<span>MusicApp</span>}
/>
<Footer variant="mini" className="bg-gray-950/95 border-gray-800">
  <div className="flex items-center gap-3">
    <span>Track Title</span>
    <button><Play /></button>
  </div>
</Footer>
```

## Dashboard

An admin panel with a fixed dark header (including mobile menu) and a static tab bar.

- **Header**: `behavior="fixed"`, `theme="dark"`, with `mobileMenu`
- **Footer**: `variant="tab-bar"`, `behavior="static"`

```tsx
<Header
  behavior="fixed"
  theme="dark"
  logo={<span>AdminPanel</span>}
  mobileMenu={
    <nav className="flex flex-col gap-1">
      <button>Dashboard</button>
      <button>Analytics</button>
      <button>Settings</button>
    </nav>
  }
/>
<Footer variant="tab-bar" behavior="static" className="bg-gray-950/95">
  <FooterItem icon={<LayoutDashboard />} label="Dashboard" active />
  <FooterItem icon={<BarChart3 />} label="Analytics" />
  <FooterItem icon={<Users />} label="Users" />
  <FooterItem icon={<Settings />} label="Settings" />
</Footer>
```
